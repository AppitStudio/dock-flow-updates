What's New in DockFlow 1.13

Major New Feature

  • DockFlowCLI
    • New command-line interface for advanced users and integrations
    • Control DockFlow directly from Terminal
    • Apply any preset via simple command-line commands
    • Get a list of available presets with IDs for easy reference
    • View currently active dock items
    • Access the command line running: /Applications/DockFlow.app/Contents/MacOS/DockFlowCLI --help

Core Improvements

  • Enhanced Core Architecture
    • Major refactoring of core functionality for better stability and performance
    • Smart detection of dockutil availability for improved dock manipulation
    • Fallback mechanisms using defaults command when needed
  • Advanced Dock Item Management
    • Enhanced folder stack manipulation with detailed settings control
    • Improved spacer handling with distinct types (regular and small)
    • Better path handling with robust URL encoding/decoding

CLI Commands

  • dockflowcli list - Display all available presets with their IDs
  • dockflowcli apply --name "Preset Name" - Apply a preset by name
  • dockflowcli apply --id 5 - Apply a preset by its ID number
  • dockflowcli get-apps - List all apps currently in your Dock

This update lays the groundwork for future integrations with popular macOS productivity apps. Stay tuned for more exciting developments in the DockFlow ecosystem!
