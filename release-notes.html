What's New in DockFlow 1.4

Major New Feature

  • Dock Spacers Support
    • Added support for both regular and small spacers in dock layouts
    • Ability to capture existing spacers when importing current dock layout
    • New dedicated buttons to add spacers directly from DockFlow's interface
    • Full support for saving and restoring spacers in presets

UI Enhancements

  • Improved App Management
    • Streamlined app addition process with a single "Add App" button
    • Clear visual separation between app and spacer controls
    • Enhanced button layout with app management on the left and spacer controls on the right
    • Added a plus icon to the "Add App" button for better visual clarity

Technical Improvements

  • Enhanced URL Processing
    • Better handling of paths with spaces
    • Proper URL encoding/decoding
    • Improved file existence validation
  • Implemented a more robust system for tracking apps in the dock
  • Added detailed logging for easier troubleshooting

Other Improvements

  • Better error messages for invalid apps
  • Improved validation of app paths before adding to presets
  • More consistent handling of dock items
  • Enhanced preset saving and loading reliability

This update introduces spacer support to DockFlow, making it a complete solution for managing your dock layout, along with various usability improvements.
